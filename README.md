# 3D Shooter

Эта игра — результат эксперимента, в котором три нейросети (Claude, GPT, Gemini) создавали 3D-шутер с нуля.

Подробности в видео: https://youtu.be/DLqXprOOvj8

## Технологии

- TypeScript + Vite
- Процедурная генерация графики (спрайты, текстуры — всё рисуется кодом)
- Raycasting-рендер

## Запуск

### Онлайн

https://studioigor.github.io/shooter-claude-gpt-gemini/

### Локально

```bash
npm install
npm run dev
```
